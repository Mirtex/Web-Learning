Content structure implementations:

The following section goes through the structure tags implemented in this document.

This document contains the following content structure tags implemented:

<nav> tag

This tag was used to mark the html document navigation.

<article> tag

This tag was used to mark the main article section of this specific document.

<section> tag

This tag was used to separated the descriptions of each html file.

<aside> tag

This tag was used to mark the Special character notes given that is tangential information.

<div> tag

This tag was used in this section, to tag each individual piece. There might be better tags for this however.
In this case I am grouping them together due to future - fictional - CSS usage (As stated above, this is HTML only.)

<header> tag

This tag was used to mark the header of the body.

<footer> tag

This tag was used to mark the footer of this page, which containes personal information about
the creator of these documents.

<main> tag

This tag was used to mark the main section of this document, which is simply the article.
This tag is used primarily of assistive devices, which is why it's important, even if it's being
overlayed around the article tag.
