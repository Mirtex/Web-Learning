Special Characters testing ground

Special characters are written with &, so even "&" cannot be written by itself.

    Some other common special characters are written in the following way:
    &lang;  - ⟨
    &rang;  - ⟩
    &lt;    - <
    &rt;    - >
    &amp;   - &
    &nbsp;  - (white space)

Notes

To format text appropiately, the correct tool is CSS. We can achieve this through HTML, but it's not meant to. For example, the white space character is used for situations like a name you would want to stick together and not go over to the next line. (Like this phrase, for example, will always stay in one line together.)
